rEFInd Minimal

A clean and minimal theme for rEFInd, bringing elegance to your boot manager

  • Source on Github
  • Install Theme
  • Evan Purkhiser
  • cd /boot/EFI/refind/
  • mkdir -p themes
  • git clone {{site.github.repository_url}} themes/{{site.github.repository_name}}
  • echo "include themes/refind-minimal/theme.conf" >> refind.conf
  • systemctl reboot
